# 持仓记忆文件格式规范

## 文件命名
`memory/YYYY-MM-DD.md` — 每日一份，截图当日日期

## 必填字段（每只股票）
| 字段 | 说明 | 示例 |
|------|------|------|
| 股票名称 | 中文全称 | 阿里巴巴-W |
| 代码 | 含市场后缀 | 09988.HK |
| 账户 | 账户一/二/三 | 账户一（港股主） |
| 市值 | 含货币单位 | 609,500 HKD |
| 持仓量 | 整数 | 4,600 |
| 现价 | 数字 | 132.50 |
| 成本 | 数字 | 146.53 |
| 持仓盈亏(元) | 数字，含正负号 | -64,530 |
| 持仓盈亏% | 百分比 | -9.57% |

## 标准Markdown表格格式
```markdown
## 账户一：港股主账户

| 股票 | 代码 | 市值(HKD) | 持仓量 | 现价 | 成本 | 持仓盈亏 | 盈亏% |
|------|------|---------|--------|------|------|---------|------|
| 阿里巴巴-W | 09988.HK | 609,500 | 4,600 | 132.50 | 146.53 | -64,530 | -9.57% |
```

## 完整性校验清单
提取完成后必须核对：
- [ ] 截图共 N 只股票
- [ ] 记忆文件写入 N 条
- [ ] 两数相等，否则补充

## 账户分类
- **账户一**：港股主账户（老虎/富途）
- **账户二**：港股/美股混合账户（老虎）
- **账户三**：国金证券A股账户（同花顺）
