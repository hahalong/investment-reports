# 操作建议分析框架

## 优先级分级
| 级别 | 触发条件 | 操作时机 |
|------|---------|---------|
| P1 紧急 | 亏损>15% 且 逻辑破坏，或杠杆ETF>-20% | 周一开盘30分钟内 |
| P2 锁利/注意 | 浮盈>20% 或 亏损10-15% | 当天内完成 |
| P3 持有/观望 | 主线逻辑完好，波动<10% | 本周内灵活操作 |

## 每只股票必须包含的字段
1. **操作方向**：清仓 / 减仓 / 持有 / 加仓 / 观望
2. **止损位**：具体价格，不能为空
3. **目标价**：具体价格或"长期持有"
4. **加仓位**（如适用）：具体价格区间
5. **操作逻辑**：1-2句话，说明为什么这样操作
6. **关键催化日**（如适用）：财报、政策发布等

## 特殊情况处理规则
- **杠杆ETF**：须注明每日损耗风险，止损更严格（-10%即减仓）
- **成本价异常**（如负数或极高）：注明"可能含反向拆股"，不以此为止损参考
- **两账户持有同一标的**：合并分析，单独标注各账户持仓量
- **已清仓（持仓量=0）**：无需操作建议，注明"已清仓"

## 宏观框架对应操作方向
| 板块 | 宏观判断 | 默认倾向 |
|------|---------|---------|
| AI算力/CPO/算电协同 | 两会政策强催化 | 持有/加仓 |
| 港股科技（阿里/腾讯） | 低估值+财报催化 | 持有 |
| 黄金/黄金ETF | 去美元化+避险 | 持有/加仓 |
| 传统油气/化工 | 偏空 | 减仓/清仓 |
| 地产链 | 偏空 | 回避 |
| 杠杆ETF（非黄金） | 时间损耗风险 | 谨慎，设严格止损 |
